## 🚀 **Quick Start**

### Prerrequisitos
- Node.js 18+ 
- npm o yarn
- TypeScript 5+

### Instalación Rápida
```bash
# Clonar el repositorio
git clone [repository-url]
cd UTN-DS25-practicas-main

# API Express
cd API-CON-EXPRESS
npm install
npm run dev

# React Apps
cd ../REACT/mi-primera-app
npm install
npm run dev

# React + Tailwind
cd ../../REACT-UI-LIB/mi-primera-app-TAILWIND
npm install
npm run dev
```
